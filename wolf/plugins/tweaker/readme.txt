== WHAT IT IS ==

The googlemap plugin lets you embed Google Maps anywhere in your Frog website.

This plugin currently has no features

== HOW TO USE IT ==

* To use the settings and documentation pages, you will first need to enable 
  the plugin.

== NOTES ==

* To use the settings and documentation pages, you will first need to enable
  the plugin.

* Did you know you can have the controller make a visible tab (or not)? Change
    Plugin::addController('googlemap', 'Google Map', 'administrator', false);
  to
    Plugin::addController('googlemap', 'Google Map', 'administrator', true);
  to have the plugin controller create a tab.
